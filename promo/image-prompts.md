## Quick Screenshot Method

If you prefer real screenshots over AI-generated graphics:

1. Open each tool in browser
2. Use browser dev tools to set viewport to 1080x1080
3. Take screenshot with clean sample data loaded
4. Add text overlay in Figma/Canva with tool name + tagline
5. Apply subtle purple gradient overlay for brand consistency
