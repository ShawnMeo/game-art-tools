# Game Art SaaS Tools

This repository contains a collection of web-based tools designed for game artists and technical artists.

## Tools Included

### [Asset Naming Validator](./asset-naming-validator/)
Validates file names against project naming conventions to ensure consistency.

### [Channel Packer](./channel-packer/)
A utility to pack different grayscale images into the RGB channels of a single texture.

### [LOD Budget Calculator](./lod-budget-calculator/)
Calculates and visualizes polygon budgets for different Level of Detail (LOD) stages.

### [Texel Architect](./texel-architect/)
Helps plan and calculate texel density for 3D assets to maintain consistent texture resolution.

### [Trim Sheet Planner](./trim-sheet-planner/)
Assists in planning and laying out trim sheets for optimized environment texturing.

## Usage
Open `index.html` in your browser to access the main dashboard with links to all tools.

## Development
Each tool is contained in its own subdirectory. They are standard web projects (mostly Vite-based).
